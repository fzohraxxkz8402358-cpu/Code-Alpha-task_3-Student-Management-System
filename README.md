# 🎓 Student Management System in C

A **menu-driven Student Management System** developed in **C Programming**. This project demonstrates the use of **structures**, **file handling**, and **modular programming** to manage student records efficiently through a command-line interface.

## 📌 Features

* ➕ Add Student Record
* ✏️ Update Student Record
* ❌ Delete Student Record
* 🔍 Search Student Record
* 📋 Display All Student Records
* 💾 Store Records Permanently Using File Handling

## 🛠️ Technologies Used

* C Programming
* GCC Compiler (MinGW)
* Visual Studio Code

## 🚀 Getting Started

### Prerequisites

* GCC Compiler
* Visual Studio Code or any C IDE

### Compile

```bash
gcc student_management.c -o student_management
```

### Run

**Linux/macOS**

```bash
./student_management
```

**Windows**

```bash
student_management.exe
```
